# Clyde

A command-line interface client for Discord.

## Installing

1. Install [Node.js 8+](https://nodejs.org/en/download/current/)
2. Clone the repository
3. Navigate to the directory
4. Run `npm install`
5. Edit `src/config.json` with your User token
6. Run `npm start`
